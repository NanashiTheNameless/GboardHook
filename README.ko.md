# GboardHook-NanashiTheNameless

**모든 크레딧은 <https://github.com/chenyue404/GboardHook>에 돌아갑니다. 이것은 개인 용도로 수정 및 번역된 포크입니다**

Gboard 클립보드에 표시되는 항목 수와 보존 시간을 수정합니다

enable_clipboard_entity_extraction 및 enable_clipboard_query_refactoring은 읽을 수 있는 클립보드 항목의 수에 영향을 미칩니다. 원리가 명확하지 않고 핵심 코드가 아직 발견되지 않았으므로, 현재로서는 이러한 설정을 false로 하드코딩할 수밖에 없습니다
