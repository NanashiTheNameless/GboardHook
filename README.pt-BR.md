# GboardHook-NanashiTheNameless

**Todos os créditos vão para <https://github.com/chenyue404/GboardHook>. Este é um fork modificado e traduzido para meu uso pessoal**

Modifica a quantidade de itens da área de transferência do Gboard exibidos e seu tempo de retenção

enable_clipboard_entity_extraction e enable_clipboard_query_refactoring afetam o número de entradas da área de transferência que podem ser lidas. O princípio não é claro e o código-chave ainda não foi encontrado, portanto, estes só podem ser codificados como false no momento
