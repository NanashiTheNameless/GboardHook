# GboardHook-NanashiTheNameless

**All credit goes to <https://github.com/chenyue404/GboardHook>. This is a modified and translated fork for my personal use**

Modifies the number of Gboard clipboard items displayed and their retention time

enable_clipboard_entity_extraction and enable_clipboard_query_refactoring affect the number of clipboard entries that can be read. The principle is unclear, and the key code hasn't been found, so these can only be hardcoded to false for now
