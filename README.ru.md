# GboardHook-NanashiTheNameless

**Все заслуги принадлежат <https://github.com/chenyue404/GboardHook> Это модифицированный и переведенный форк для личного использования**

Модифицирует количество отображаемых элементов буфера обмена и время их хранения

enable_clipboard_entity_extraction и enable_clipboard_query_refactoring влияют на количество записей буфера обмена, которые можно прочитать. Принцип неясен, и ключевой код еще не найден, поэтому сейчас их можно только закодировать как false
